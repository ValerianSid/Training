Приложение для тренировки в спортзале

Роли:
-Администратор
-Тренер
-Пользователь

Сущности:

	User:
	-id
	-login
	-password
	-name
	-List<role>
	-List<Личная тренировачная программа_id>

	Упражнение:
	-id
	-name
	-discription
	-image

	Тренировачная программа:
	-id
	-name
	-discription
	-List<Упражнения_id>

	Личное упражнение(наследуется от упражнение):
	-id
	-кол-во подходов
	-вес
	-упражнение_id

	Личная тренировачная программа():
	-id
	-name
	-discription
	-List<Личное упражнение_id>

	Завершенная тренировка:
	-id
	-Личная тренировачная программа_id

Функции по ролям:

	Администратор:
	-создание user с ролями администратора, тренера, пользователя
	-создание упражнения
	-создание тренировки
	
	Тренер:
	-создание user с ролями пользователь
	-создание личной тренировачной программы

	Пользователь:
	-создание личной тренировачной программы

Методы:

	-создание user с ролями
	-создание упражнения
	-создание тренировки
	-создание личной тренировки
	-создание завершенной тренировки
